# Controlled Components
- Basically inputs (any kind) who's data is tracked as state. 
If 'value' is used as a prop with an input, its controlled and so cannot be updated directly on the page. It must be synchronized with a component state.

> You provided a `value` prop to a form field without an `onChange` handler. This will render a read-only field. If the field should be mutable use `defaultValue`. Otherwise, set either `onChange` or `readOnly
