# Async vs. Sync
- In Java, an https request is synchornous and the program will halt to wait for the request to complete. 
- In javascript, which is async, the code keeps executing as the https request (I/O) occurs. 
- This is also because javascript is single threaded and if we halt for an I/O operation, the whole program becomes non-interactive until the operation finishes. See more on javascript's event loop, callback queue, stack, webAPI.
