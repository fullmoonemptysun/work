# Layout Vocabulary
1.  **Activity:** A screen in android.
2. **Layout**: The structure of a UI, such as an activity.
3. **XML:** Used to define layouts
4. **View:** component of a layout.


# ConstraintLayout
1. Most popular android layout due to its:
	- flexibility
	- performance
	- visual layout editor

2. Allows us to easily develop responsive UIs that adapt seamlessly to different screen sizes.
3. Describes Constraints among views.

# Common Views:
1. TextView: Displays text
2. ImageView: displays image
3. Button: spawns a button
4. EditText: allows for user input and modify text
