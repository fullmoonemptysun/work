
- RGB = R x 65536+G x 256+B
- Two ways to measure image quality:
  - Bit depth (BPP number of color bits per pixel)
  - Resolution(DPI dots per inch / PPI pixels per inch)



- Data compression: ● Coding of data from a larger to a smaller form. ● Types: ● Lossless compression and lossy compression
	- Lossless: Redundant bits. 200 white bytes => 2 bytes: 1 byte what color, 1 byte how many. (Huffman/Lempel). (exact replica).
	- Lossy: Will be an altered (some non-critical parts lost) version.


## RAW file (Camera graphics format)
- aka digital negative (DNG)
- Purest, rawest form of a picture no processing occurs.
- Can be augmented and processed later so has a big size.
- Captures all data from the sensors.
- process of converting from this format to others is called demosaicing/ color Reconstruction


## EXIF
- Used to store metadata in most digital photographs (JPEGs, TIFs)

## ELA Error level analysis
- Basically any newly added areas will not have the same compression rate as the original image. And so resaving and compression will reveal them.
