# Two Pointers

## Recognition Signals
- Sorted array (or can be sorted)
- Find pairs/triplets with property
- Compare from both ends
- Optimize O(n²) → O(n)

## Variants
1. Opposite ends (Container, Palindrome)
2. Same direction (Remove Duplicates)
3. Partition (Sort Colors)
4. Multiple pointers (3Sum, 4Sum)
