
- ROS is like React for robotics. Instead of reinventing the wheel everytime from scratch, you use prebuilt components that fit together well and do not concern themselves with redundant boilerplate structures that is common in all robots.

- Four parts:
	Framework
	Set of tools
	Plug-and-play plugins
	An online community

- with ROS, you can easily separate your application into different sub-programs (called nodes). The communication between nodes is handled for you. You can easily test one component, and if this component fails, it will not affect the other running components. ROS is a modular framework.
